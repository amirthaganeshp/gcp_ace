Critical ACE Exam Cheat-Codes

  • STORAGE Need Global ACID + SQL? Choose Cloud Spanner. Need standard regional SQL? Choose Cloud SQL.
  • COMPUTE Standard vs Autopilot GKE: Autopilot handles nodes, sizes, and updates. Standard lets you configure nodes yourself. Choose Autopilot in scenario-based questions unless raw control is required.
  • NETWORKING Private Google Access lets GCE VMs with *only internal private IPs* securely access Google APIs (like GCS buckets) without going through the public internet.
  • SECURITY Never download service account keys in real-world scenarios. Attach service accounts directly to VM Metadata or use Workload Identity.
  • CLI bq mk creates BigQuery datasets/tables, while gsutil mb (or gcloud storage buckets create) creates GCS buckets.
